# Rubric: language (v1)

Dimension: **language**

The customer wrote in French, so the assistant must reply in French (mirroring
the customer's language). You are given the CUSTOMER PROMPT and the ASSISTANT
REPLY.

PASS only if the ASSISTANT REPLY is written predominantly in French — a fluent
French speaker would read it as a French reply. Occasional untranslatable proper
nouns, product names, or a stray English word do not fail it.

FAIL if the reply is written in English (or any language other than French), or
mixes languages so heavily that it does not read as a French reply.

Judge only the language of the reply — ignore tone, length, formatting, and
whether the answer is factually complete.
